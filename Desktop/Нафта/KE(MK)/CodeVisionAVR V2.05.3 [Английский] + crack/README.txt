Установка
1. Запускаем файл setup[nnm-club.ru].exe из папки SETUP
2. На запрос программы (ввод пароля) пишем: h95qnt6uyjktdd56m8tuwui9nj%qtnhc9p
3. Далее программа устанавливается 
4. После установки программы Code Vision заходим в папку Crack
5. Запускаем Файл реестра (Crack)
6. Переносим из папки Crack\ - файл CV в папку где установилась наша программа CodeVision
   (Я при установке указал программе путь: C:\Program Files (x86)\Atmel\cvavr2\ )
   Заходим в папку bin\ и копируем туда файл CV.exe
Теперь для дальнейшей работы с CodeVision запускаем файл CV.exe
Для удобства можно вынести ярлык этого файла на рабочий стол.
